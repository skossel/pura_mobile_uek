# Atemglanz Pura App (MVP)

## 📱 Projektübersicht

Die Atemglanz Pura App ist eine mobile React-Native-App (Expo), die als Prototyp für eine zukünftige Smart-Funktion eines Luftreinigers dient.

Aktuell werden die Luftreiniger-Daten simuliert. Die App kombiniert diese mit Smartphone-Kontext (z. B. Zeit, optional Standort), um verständliche Einblicke und Empfehlungen zur Raumluft zu geben.

Der Fokus liegt auf:
- einfacher Bedienung
- klarer Darstellung
- mobile-first UX
- Vorbereitung auf spätere Hardware-Integration

---

## 🎯 Zielgruppe

- Allergiker
- gesundheitsbewusste Nutzer
- Personen, die ihre Raumluft verbessern möchten
- Nutzer, die einfache Lösungen ohne technische Komplexität bevorzugen

---

## 💡 Hauptfunktionen (MVP)

- Dashboard mit Luftqualität (simuliert)
- Anzeige von PM2.5, Temperatur, Luftfeuchtigkeit, Filterstatus
- verständliche Statusbewertung („gut“, „belastet“)
- intelligente Empfehlungen (z. B. basierend auf Tageszeit)
- Modus-Steuerung (Auto, Sleep, Boost – simuliert)
- Verlauf der Luftqualität (simuliert)
- Demo-Modus Hinweis
